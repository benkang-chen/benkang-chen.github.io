hexo 基本使用

1. 新建仓库

新建一个github 仓库，仓库名必须是.github.io 格式，其中 是你 github 的昵称。

2. 全局安装hexo

3. 初始化项目

新建目录，在新建目录下，执行 hexo init ，就会自动构建一个 hexo 项目，紧接着执行 hexo s，浏览器访问 localhost:4000 就可以看到效果。

4. 部署到github

在项目根目录下找到 _congif.yml，找到 deploy 字段并填写完整。

推送到仓库需要安装一个额外的工具。

部署

5. 查看效果

访问自己的github io 地址即可查看到效果。

6. 新建文章

在 source/post找到新建文章的md文档，可添加tags， categories

7. 创建标签与分类页

创建标签页

配置标签页，在source下找到对应的tags目录，配置index.md文档

配置分类页，在source下找到对应的categories目录，配置index.md文档

8. 部署发布新文档

9. 删除文章

删除文章，直接删除md文档，但要保证至少有一个md文件。然后执行 hexo clean.
